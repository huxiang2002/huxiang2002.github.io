北京邮电大学考研机试题——二进制

算法 位运算
算法
发布日期:   2023-08-10

题目描述

给定一个长度为 位的二进制数 ，请你计算并输出 和 的二进制表示结果。

注意，结果不能忽略前导 （不够 位的用前导 补足 位，超过 位的不用补前导 ）。

输入格式

第一行包含整数 ，表示共有 组测试数据。

每组数据占一行，包含一个长度为 的 01 字符串。

输出格式

每组数据输出两行，第一行为 的二进制表示结果，第二行为 的二进制表示结果。

数据范围

1≤≤100

输入样例：

2
00000000000000000000000000000000
00000000000000000000000000000001

输出样例：

00000000000000000000000000000001
00000000000000000000000000000011
00000000000000000000000000000010
00000000000000000000000000000100

int的数量级是，long long 的数量级是。我们完全可以使用long long来存储32位二进制数转化的10进制数。在10进制数+1，+3再转化为二进制数输出即可。

如果二进制n全为1的话+1，+3最多会进1位。所以我们从32位枚举即可

代码如下

#include<iostream>
#include<cstring>
using namespace std;

typedef long long LL;

void print(LL n)
{
    for(int i=32;i>=0;i--)
    {
        int t=n>>i&1;// 取t的每一位的二进制数
        if(!t&&i==32) continue;//第33位的数为0 不输出
        cout<<t;
    }
    cout<<endl;
}

int main()
{
    int t;
    cin>>t;
    while(t--)
    {
        LL n=0;
        string s;
        cin>>s;
        for(int i=0;i<s.size();i++)
            n=n*2+s[i]-'0';//将二进制数转化为10进制数
        print(n+1);
        print(n+3);
    }
    return 0;
}
